Frequently Asked

A .sol subdomain is a readable name under an existing .sol domain, such as yourname.mytag.sol. It can resolve to a Solana wallet address in compatible wallets and apps, so people can send to a name instead of a long public key.

Registration links open the official Solana Name Service interface. Wallet connection, checkout, terms, and transaction confirmation happen there, not on this directory page.

The listed subdomain registrars are currently presented as one-time registrations with no annual renewal flow. This depends on current protocol behavior and registrar configuration, which may change outside this page's control.

Prices for listed registrars currently start from 3 USDC for longer names. Shorter names may cost more. SNS may also charge platform fees, and every registration requires a Solana network transaction fee. Always review the final amount on sns.id before confirming.

SNS subdomains can support transfer and record updates, but they are not identical to root .sol domains. SNS documentation notes that parent-domain and registrar rules can affect subdomain behavior, and wallet support may be more limited than for root .sol names.

Many Solana wallets and apps support .sol name resolution, including widely used wallets such as Phantom, Solflare, and Backpack. Support can differ between root domains and subdomains, so test the exact name in the wallet or app before relying on it for payments.
