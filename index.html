Got questions?

Frequently Asked

A .sol subdomain is a readable Solana Name Service name under an existing .sol domain, such as yourname.mytag.sol. It can point to on-chain data, especially a Solana wallet address, so compatible apps can use a name instead of a long public key. Names are first come, first served within each namespace.

SNS names are built for payments, profiles, identity, and dApps across Solana. The official integration list includes widely used wallets and explorers such as Phantom, Solflare, Backpack, Trust Wallet, Solscan, and more. Support can differ by app, and subdomain support may differ from root .sol names, so test the exact name before relying on it for payments. See the SNS integration list and SNS usage docs.

Listed registrars currently start from 3 USDC for eligible longer names, plus the SNS fee and Solana transaction fee. Some registrars may use length-based or namespace-specific pricing. Registrations are currently presented as one-time purchases with no annual renewal flow, but pricing, tiers, fees, and registrar settings can change. Always confirm the final amount on sns.id before purchasing.

After checkout, the subdomain is assigned to your wallet as an on-chain SNS subdomain. Where supported, it can be used, transferred, and updated through SNS-compatible tooling. It is not the same as buying a root .sol domain: parent-domain and registrar rules still matter, and wallet support can be limited. The spelling is fixed, so if you want a different name or namespace later, register another one.

Every registration link opens an external Solana Name Service page. Wallet connection, checkout, terms, and transaction confirmation happen there, not on this page.

Before purchasing, verify the exact spelling, availability, final price, SNS fee, Solana transaction fee, allowed characters, registrar settings, wallet or app support, and subdomain limitations on SNS. mytag.sol is an independent information and link hub, not an official SNS, Solana, wallet, or token-project website. Use the SNS subdomain docs for protocol details.
